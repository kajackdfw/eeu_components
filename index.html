eeu_html_bootstrap.html web components
Web Component Description Status IE11 Firefox 39.0 Chrome 44
***-dec-to-frac An input field with a informative label that gives user feedback of feet, inches, and fraction translation of their decimal input value. The name parameter is used for a hidden field with a rounded 64ths value, so value in database/post submit will always match the fraction even when the user does not know the exact decimal equivalent of a fraction. Working Yes Yes Yes
***-dec-to-frac-ro A readonly display of a decimal to feet, inches, and fraction conversion. Working Yes Yes
***-double-select A select box that asks for a type or category first, then displays a list of options. Working Throws error #184 Yes Yes
***-string A string input field with min and max character validation. Working Yes
***-string-inline A string input field with min and max character validation with bootstrap inline style. This function may become eeu-string with a fstyle="inline" param later. Working Yes
***-fifrac-to-dec An input field that allows feet inches and fraction text to be entered, and a decimal equivalent is submitted to app on submit.
***-pattern This is a free text field with a displayed format pattern like a phone number (817)281-7673 EXT 2345, and a matching filter on the back, so a raw number 81728176732345 can be stored in the db. This will allow auto formatting for raw data coming from the db. [999] = numeric , [AAA] for alphabetic
***-unix-date A datepicker with desired user date pattern on front, but always submits a unix value.
***-rest-validated-input A free text field with a ajax validation built in.
***-dropdown A drop down with values and labels from a rest service
***-combo Similar to the eeu-input-rest-validated, but with an additional drop down of predefined values.
***-submit A submit button that is disabled if any eeu fields are transmitting an invalid signal.
