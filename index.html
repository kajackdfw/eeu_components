Web Component Description Status IE11 Firefox 39.0 Chrome 44
eeu-dec-to-frac An input field with a informative label that gives user feedback of feet, inches, and fraction translation of their decimal input value. The name parameter is used for a hidden field with a rounded 64ths value, so value in database/post submit will always match the fraction even when the user does not know the exact decimal equivalent of a fraction. Working Yes Yes Yes
eeu-dec-to-frac-ro A readonly display of a decimal to feet, inches, and fraction conversion. Working Yes Yes
eeu-string A string input field with min and max character validation. Working Yes
eeu-fifrac-to-dec An input field that allows feet inches and fraction text to be entered, and a decimal equivalent is submitted to app on submit.
eeu-phone User enters or pastes a phone string, and it is filtered, cleaned, and or reformatted to the designated pattern. This could become just a wrapper for a possible eeu-input-regex component.
eeu-unix-date A datepicker with desired user date pattern on front, but always submits a unix value.
eeu-rest-validated A free text field with a ajax validation built in.
eeu-combo Similar to the eeu-input-rest-validated, but with an additional drop down of predefined values.
eeu-submit A submit button that is disabled if any eeu fields are transmitting an invalid signal.
